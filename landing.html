Fonctionnement

Opérationnel en 48 heures.

1
Vous commandez votre pack

Nous créons votre instance dédiée : sous-domaine personnalisé, base de données isolée, accès sécurisé (HTTPS + Basic Auth). Pas de configuration de votre côté.

2
Vous importez vos données

Glissez-déposez vos exports Bizouk, Weezevent ou Weezpay. Le parseur détecte automatiquement le format, calcule le CA net et structure les données.
